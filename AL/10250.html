t = int(input())

for i in range(t):
    h, w, n = map(int, input().split())
    num = n//h + 1
    floor = n % h
    if n % h == 0:
        num = n//h
        floor = h
    print(f'{floor*100+num}')

    본 문제의 패턴은 다음과 같다.


    첫 번째 열을 1층 부터 H층 까지 채우고, 두 번째 열로 넘어가서 반복한다.

    호수는 $n/h$의 몫에 1을 더한 값이고, 층수는 나머지 값이다.

    주의할 점은 호수가 1일때, 층수는 h값과 동일하고 호수는 $n/h$ 이다.
